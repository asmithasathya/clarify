\begin{abstract}
We study intent stabilization for ambiguous user requests. The core system samples multiple structured interpretations of a request, detects unstable understanding points, selectively resamples the weakest stage, and asks a clarification question only when intent uncertainty remains high. The repository now supports the full research workflow around that method: frozen train/dev/test data splits, baseline and ablation matrices, teacher rollout export, modular student distillation, optional DPO preparation, and a living manuscript whose tables are generated directly from experiment artifacts.
\end{abstract}
